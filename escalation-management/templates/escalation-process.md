
# Escalation Process

## Levels
| Level | Description | Example |
|-------|-------------|---------|
| L1 | Low priority | Minor access issue |
| L2 | Team-level | Integration delay |
| L3 | High | Functionality gap |
| L4 | Critical | Renewal at risk or outage |

## Steps
Identify → Classify → Contain → Communicate → Resolve → Post‑mortem.
